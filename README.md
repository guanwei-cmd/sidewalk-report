## 二、部署方式

- **本機**：直接雙擊 `index.html` 即可（無需伺服器）。
- **靜態託管**：把 `index.html` 丟上任何靜態空間（GitHub Pages／Netlify／S3…）即完成部署。
- 個人化使用 `localStorage`（鍵：`npp_city`、`npp_age`、`npp_sid`），無 cookie、無第三方追蹤；預設無外部請求（字體除外）。若啟用後台統計（見 `backend/`），僅匿名送出縣市／年齡／事件。

## 三、網站結構與互動

1. **首屏遊戲「這條人行道，裝得下你的家人嗎？」**：把大人／小孩／輪椅／嬰兒車／牽繩狗／白手杖放上人行道橫斷面（購物車式加減），需求淨寬自動加總、對照法定下限 1.5m，選縣市即顯示「只有 X% 人行道淨寬走得下這一組」（資料：22 縣市淨寬累積分布，0.1m 步進、長度加權）。選擇的縣市成為**全站個人化錨點**：首屏速覽卡、後續圖表紅點「你」標記、年齡計算機預選。
2. **你來畫**：畫出你以為的普及率曲線 → 揭曉真實（0–100% 軸）。可產生 1080×1350 直式社群圖卡分享。
3. **年齡計算機**：照每年 +0.87pp，你的縣市補到八成時你幾歲。
4. **壹～陸章 scrollytelling**：全寬大圖＋隨捲動變化的置中圖說、focus 先行；敘事走完後浮現排序／口徑篩選控制列（narrative-first, exploration-second）。
5. **貳章「四類分母口徑」**：老實（全道路＝省+縣+鄉+市區）／都計・市區子集／自訂分母／狀況外與矛盾——縣市長條依口徑上色，「同一把尺」以各報告自載之全道路總長校正（桃園 54.64→11.69、臺南 39.19→19.27；未揭露總長者誠實標「無法換算」）。
6. **社群之聲**：各章引句卡均附原始連結；肆／伍章與年齡計算機附主題相關之時代力量交通專題延伸閱讀。

## 四、資料口徑備忘（重要）

- **全國普及率 39.29%**（113 年底）＝人行道 8,511.95 km ÷ **市區道路** 21,666.73 km（國土署／立院預算中心表 1）。
- **「老實／同一把尺」分母＝全道路（省道+縣道+鄉道+市區道路）**，依宜蘭・南投健檢報告定義表；各縣市能否校正取決於報告是否自載全道路總長（詳附件拾壹節）。
- 寬度／淨寬分析＝dataset 58791（2024-12，88,677 段）以 SW_WTH／SWW_WTH、SW_LENG 長度加權；淨寬分母為有填淨寬之路段；1.5m 為法定淨寬原則下限。
- 各縣市自報數與分子計法（單側／雙向）不一，圖表 ⚠ 與 tooltip 已逐縣標註。
