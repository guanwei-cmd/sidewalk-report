# 後台收資料（縣市統計）部署說明

網站是純靜態頁（GitHub Pages），後台用 **Google Apps Script（GAS）＋ Google 試算表** 承接：免費、免伺服器、資料進你自己的 Google 帳號。前端已內建發送機制（`navigator.sendBeacon`），**預設關閉**——`index.html` 裡的 `COLLECT_URL` 為空字串時完全不發送任何請求。

## 收什麼（皆匿名）

| 欄位 | 內容 |
|---|---|
| `ts` | 伺服器收到時間（GAS 端蓋章） |
| `sid` | 隨機工作階段碼（localStorage 產生，無法回推個人） |
| `ev` | 事件：`city`（首屏選縣市）/ `age`（填年齡）/ `calc`（用年齡計算機） |
| `city` | 使用者定錨的縣市（臺字正體） |
| `age` | 年齡（未填為空） |
| `need` / `pct` | 當下組隊需求寬度(m)與該縣市達標率(%) |

不收 IP 之外的任何個資（GAS 端我們也不記 IP）；無 cookie、無第三方追蹤。

## 部署步驟（約 5 分鐘）

1. 用 nppnppnpp2026@gmail.com 開一份新的 **Google 試算表**，命名如「sidewalk-report 縣市統計」。
2. 試算表選單 **擴充功能 → Apps Script**，貼上下方程式碼（取代預設內容）：

```javascript
function doPost(e){
  var sh = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('log')
        || SpreadsheetApp.getActiveSpreadsheet().insertSheet('log');
  if (sh.getLastRow() === 0)
    sh.appendRow(['ts','sid','ev','city','age','need','pct']);
  var d = {};
  try { d = JSON.parse((e.postData && e.postData.contents) || '{}'); } catch(err){}
  sh.appendRow([new Date(), d.sid||'', d.ev||'', d.city||'', d.age||'', d.need||'', d.pct||'']);
  return ContentService.createTextOutput('ok');
}
```

3. 右上 **部署 → 新增部署 → 類型：網頁應用程式**：
   - 執行身分：**我**
   - 存取權：**任何人**
   按「部署」，複製產生的網址（形如 `https://script.google.com/macros/s/AKfycb.../exec`）。
4. 打開 `index.html`，搜尋 `COLLECT_URL`，把網址貼進引號：
   ```js
   const COLLECT_URL='https://script.google.com/macros/s/AKfycb.../exec';
   ```
5. commit + push。之後每次使用者在首屏選縣市／填年齡／用計算機，試算表 `log` 分頁就多一列。

## 技術備註

- `sendBeacon` 以 `text/plain` 送出 → 不觸發 CORS preflight，GAS 可直接收；頁面關閉中也能送達。
- 想改收集事件：前端統一走 `collect(ev, extra)`，加一行即可。
- 若要公開透明，可在網站 footer 加一句「本站匿名記錄縣市選擇供研究統計」——啟用 COLLECT_URL 時建議一併加上。
- 分析：試算表直接樞紐（縣市 × 事件），或 File → 匯出 CSV 丟回 data-room。
